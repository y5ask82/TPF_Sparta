  # 프로젝트 소개
### 프로젝트 명 
**Trump vs Chess**
### 프로젝트 소개  
사악한 Trump 왕국이 착한 Chess 왕국에 쳐들어 옵니다.<br/>
서로 전쟁을 하는 와중에 플레이어가 체스 왕국을 돕기 위해 나섰습니다.<br/>
플레이어가 되어 체스왕국을 도와 트럼프왕국을 물리치세요!
### 게임유형 
2D 타워 디펜스 게임
### 개발기간 
2023.09.25 ~ 2023.10.04
